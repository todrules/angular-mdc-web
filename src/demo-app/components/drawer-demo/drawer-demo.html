Drawers

The MDC Drawer component is a spec-aligned drawer component adhering to the Material Design navigation drawer pattern. It implements permanent, persistent, and temporary drawers.
Component
mdc-drawer
Properties Description
drawer: string Set the drawer implementation. (Default is 'permanent') 
Valid values: 'permanent' | 'persistent' | 'temporary'
fixed: boolean Makes permanent drawer fixed on top and have persistent elevation.
Component
mdc-drawer-persistent
Method Description
open() Opens the drawer.
close() Closes the drawer.
isOpen(): boolean Returns whether or not the drawer is open.
Events Description
opened Event dispatched on drawer open.
closed Event dispatched on drawer close.
Component
mdc-temporary-drawer
Properties Description
absolute: boolean If true, drawer can open from container element (e.g.: mdc-content). (Default: false)
closeOnClick: boolean If false, clicking on the drawer will not close it. (Default: true)
Method Description
open() Opens the drawer.
close() Closes the drawer.
isOpen(): boolean Returns whether or not the drawer is open.
Events Description
opened Event dispatched on drawer open.
closed Event dispatched on drawer close.
Child Directives Description
mdc-drawer-spacer Optional. Add to provide the matching amount of space for toolbar.
mdc-drawer-header Optional. Add to create a 16:9 drawer header.
mdc-drawer-header-content Optional. Add content inside mdc-drawer-header.
mdc-drawer-content Optional. Should be set on the list of items inside the drawer.
